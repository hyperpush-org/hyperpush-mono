Add a new `mesh_query_join_as` extern C function (5 args: q, join_type, table, alias, on_clause) that encodes the join as `ALIAS:TYPE:table:alias:on_clause` in the join_clauses list. This distinguishes aliased joins from regular joins in the SQL builder.

```rust
/// Add a JOIN clause with table alias.
///
/// `Query.join_as(q, :inner, "projects", "p", "p.id = issues.project_id")` -> INNER JOIN projects p ON ...
#[no_mangle]
pub extern "C" fn mesh_query_join_as(
    q: *mut u8,
    join_type: *mut u8,
    table: *mut u8,
    alias: *mut u8,
    on_clause: *mut u8,
) -> *mut u8 {
    unsafe {
        let new_q = clone_query(q);
        let jt_str = mesh_str_ref(join_type);
        let tbl_str = mesh_str_ref(table);
        let alias_str = mesh_str_ref(alias);
        let on_str = mesh_str_ref(on_clause);
        let jt_sql = atom_to_join_type(jt_str);
        let join = format!("ALIAS:{}:{}:{}:{}", jt_sql, tbl_str, alias_str, on_str);
        let join_mesh = rust_str_to_mesh(&join);
        let jc = query_get(new_q, SLOT_JOIN);
        query_set(new_q, SLOT_JOIN, mesh_list_append(jc, join_mesh as u64));
        new_q
    }
}
```

**In `crates/mesh-rt/src/db/repo.rs`:**

Update ALL THREE join-processing loops (in `build_select_sql_from_parts`, `build_count_sql_from_parts`, and `build_exists_sql_from_parts`) to handle both encoding formats:
- Existing: `TYPE:table:on_clause` -> `TYPE JOIN "table" ON on_clause`
- New: `ALIAS:TYPE:table:alias:on_clause` -> `TYPE JOIN "table" alias ON on_clause`

The ALIAS: prefix distinguishes the new format. In each loop, check for `ALIAS:` prefix first, then fall back to existing `splitn(3, ':')` logic.

```rust
// In the JOIN clause loop:
for join in join_clauses {
    if let Some(rest) = join.strip_prefix("ALIAS:") {
        let parts: Vec<&str> = rest.splitn(4, ':').collect();
        if parts.len() == 4 {
            sql.push_str(&format!(
                " {} JOIN {} {} ON {}",
                parts[0],          // join type (INNER, LEFT)
                quote_ident(parts[1]),  // table name
                parts[2],          // alias (unquoted, like p or ak)
                parts[3]           // on clause
            ));
        }
    } else {
        let parts: Vec<&str> = join.splitn(3, ':').collect();
        if parts.len() == 3 {
            sql.push_str(&format!(
                " {} JOIN {} ON {}",
                parts[0],
                quote_ident(parts[1]),
                parts[2]
            ));
        }
    }
}
```
