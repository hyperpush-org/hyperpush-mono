Add `pub type` cross-module export support: type aliases marked `pub` are included in `ExportedSymbols` and `ModuleExports`, allowing importing modules to use the alias transparently.

Output: Cross-module type alias export/import works end-to-end; E2E fixture files, a test function in e2e.rs, and a unit test in structs.rs confirm pub alias is usable in importing module.

    if let Item::TypeAliasDef(alias_def) = &item {
        if let Some(name) = alias_def.name().and_then(|n| n.text()) {
            if alias_def.visibility().is_some() {

}
