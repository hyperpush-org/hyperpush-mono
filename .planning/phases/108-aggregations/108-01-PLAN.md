2. `mesh_query_select_count_field(q: *mut u8, field: *mut u8) -> *mut u8` -- Appends `RAW:count("field")` to select_fields (reads field as mesh string, quotes it).

3. `mesh_query_select_sum(q: *mut u8, field: *mut u8) -> *mut u8` -- Appends `RAW:sum("field")` to select_fields.

4. `mesh_query_select_avg(q: *mut u8, field: *mut u8) -> *mut u8` -- Appends `RAW:avg("field")` to select_fields.

5. `mesh_query_select_min(q: *mut u8, field: *mut u8) -> *mut u8` -- Appends `RAW:min("field")` to select_fields.

6. `mesh_query_select_max(q: *mut u8, field: *mut u8) -> *mut u8` -- Appends `RAW:max("field")` to select_fields.

Implementation pattern for each (e.g., select_sum):
```rust
#[no_mangle]
pub extern "C" fn mesh_query_select_sum(q: *mut u8, field: *mut u8) -> *mut u8 {
    unsafe {
        let new_q = clone_query(q);
        let field_str = mesh_str_ref(field);
        let raw = format!("RAW:sum(\"{}\")", field_str.replace('"', "\"\""));
        let raw_mesh = rust_str_to_mesh(&raw);
        let sf = query_get(new_q, SLOT_SELECT);
        query_set(new_q, SLOT_SELECT, mesh_list_append(sf, raw_mesh as u64));
        new_q
    }
}
```

For `mesh_query_select_count` (no field):
```rust
#[no_mangle]
pub extern "C" fn mesh_query_select_count(q: *mut u8) -> *mut u8 {
    unsafe {
        let new_q = clone_query(q);
        let raw = rust_str_to_mesh("RAW:count(*)");
        let sf = query_get(new_q, SLOT_SELECT);
        query_set(new_q, SLOT_SELECT, mesh_list_append(sf, raw as u64));
        new_q
    }
}
```
